How this works. Brackets & scores come from ESPN's public feed. Player insights are computed from the crowdsourced Match Charting Project and queried in your browser with DuckDB-WASM — nothing but this page's code is on a server. The win probability is an experimental model from each player's charted serve+return rates; it ignores surface, form, and injuries, and is only as good as the charting behind it.
